# Scobo

A simple scoreboard application made with [Solid](https://www.solidjs.com).

![Screenshot](./screenshot.png)

## Installation

```bash
    git clone https://github.com/kris-ms/scobo
    cd scobo
    bun install
    bun run dev # or bun run build
```
